## 🛠️ The Software Forge

Our projects are divided into two core pillars: **Game Technologies** and **Enterprise Solutions**.

### 🔹 1. Game Tech & Engines
* **`entropy-engine`** *(Core Technology)*
    * **Language:** C++ / Vulkan
    * **Goal:** A high-fidelity, data-oriented game engine built for massive scale. Unlike commercial engines, Entropy provides low-level control over memory and threading for unparalleled performance.
    * **Use Case:** Powering our internal game projects and serving as the simulation backbone for [NullAsh Robotics](https://github.com/NullAsh-Robotics).

* **`hisoka-system`**
    * **Language:** C++ / Lua
    * **Goal:** An emergent narrative and state-machine framework. It allows NPCs to develop complex, evolving traits based on player interaction, moving beyond static dialogue trees.

### 🔹 2. SaaS & Developer Tools
* **`CodeReviewsAI`**
    * **Language:** C# (.NET 8) / Azure AI
    * **Goal:** An intelligent code analysis agent that integrates directly into CI/CD pipelines. It goes beyond linting to provide semantic feedback, security auditing, and architectural suggestions.
    * **Status:** *Active Development*

* **`offline-resilience-kit`**
    * **Language:** TypeScript / Angular / PWA
    * **Goal:** A robust framework for building "Local-First" web applications. Designed for industrial and remote environments where internet connectivity is unstable or non-existent.
